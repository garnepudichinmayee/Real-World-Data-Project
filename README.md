# Real-World-Data-Project

## Retail Sales Analysis using Python

### Project Overview
This project analyzes a real-world retail sales dataset using Python. It includes data cleaning, exploratory data analysis (EDA), data visualization, and a simple machine learning model to predict sales.

### Dataset
- **Dataset:** Superstore Sales Dataset
- **Domain:** Retail

### Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

### Project Workflow
1. Data Loading
2. Data Cleaning
3. Handling Missing Values
4. Exploratory Data Analysis (EDA)
5. Data Visualization
6. Sales by Category Analysis
7. Sales by Region Analysis
8. Profit by Category Analysis
9. Monthly Sales Trend Analysis
10. Sales Prediction using Linear Regression
11. Model Evaluation

### Visualizations
- Count of Products by Category
- Total Sales by Category
- Total Sales by Region
- Total Profit by Category
- Monthly Sales Trend

### Machine Learning
- Algorithm: Linear Regression
- Evaluation Metrics:
  - R² Score
  - Mean Absolute Error (MAE)

### Key Findings
- Technology generated the highest sales and profit.
- The West region recorded the highest sales.
- Sales showed an increasing trend over time.
- The Linear Regression model was evaluated using R² Score and MAE.

### Conclusion
This project demonstrates how data science techniques can be applied to a real-world retail dataset. It covers data preprocessing, visualization, business insights, and predictive modeling using Python.
